GET /devices/:id

Returns extended information for a given device identified from its ID. Learn more on Dev Center.

  • id - A String that represents the ID of the desired device.
  • callback - The callback function returning the results. An error object is passed as first argument and the result as last.
 
  function find(id, callback) {
    core.api('GET', '/devices/' + id, {}, callback);
  }
